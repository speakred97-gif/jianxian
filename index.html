🔑 设置 Anthropic API Key

本应用在您的浏览器中直接调用 Anthropic API，无需后端服务器。
API Key 仅保存在本地 localStorage，不会上传到任何服务器。
每次生成消耗的是您自己账户的 Token 配额，与 Claude.ai 订阅共用。

没有 API Key？前往 console.anthropic.com 获取。
